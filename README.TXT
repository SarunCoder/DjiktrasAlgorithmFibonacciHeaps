========================
BUILD OUTPUT DESCRIPTION
========================

To run the project from the command line, go to the folder containing the Dijikstra.jar file and type the following:

---------------------------------------------
java -jar "Dijikstra.jar" -f file-path

fibonacci scheme

---------------------------------------------
java -jar "Dijikstra.jar" -s file-path

simple scheme

---------------------------------------------
java -jar "Dijikstra.jar" -r n d x

Random mode

n number of vertices
d density of the graph
x source node
